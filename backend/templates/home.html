Sistema de evaluación crediticia

01 — Descripción general

¿Qué es BarboBank?

BarboBank es una simulación de un sistema de aprobación de créditos que evalúa solicitudes a partir de variables financieras y personales. Su objetivo es mostrar de forma práctica cómo un sistema puede tomar decisiones automatizadas combinando lógica de negocio tradicional con modelos de Machine Learning.

02 — Funcionamiento

¿Qué información necesita?

El usuario proporciona datos personales y financieros. A partir de ellos, el sistema calcula internamente el score crediticio y el ratio de endeudamiento — sin que el usuario los ingrese directamente.

Edad Salario Gastos actuales Historial de retrasos Meses como cliente Score crediticio (interno) Ratio de endeudamiento (interno)

03 — Lógica de decisión

¿Cómo decide el sistema?

Un modelo de Machine Learning calcula la probabilidad de incumplimiento de pago (default). Esta probabilidad define si el crédito se aprueba o rechaza.

Aprobado

≤ 40%

Probabilidad de default — crédito otorgado

Rechazado

> 40%

Probabilidad de default — solicitud denegada

04 — Enfoque investigativo

Un proyecto exploratorio

Este proyecto no busca replicar exactamente un banco real. Simplifica el proceso de evaluación crediticia para entender cómo funcionan los sistemas de decisión automatizados, el uso de ML en backend e integracion de Agentes de IA

Reconocimiento

Limitaciones conscientes

Los bancos reales usan muchos más parámetros y controles de verificación adicionales.

Objetivo

Aprendizaje práctico

Permite entender cómo se integran modelos de ML en flujos de decisión real.

05 — Agente de IA

Tu asesor financiero conversacional

Después de completar la solicitud, el sistema habilita un agente de IA que accede a la base de datos mediante la cédula del usuario. Puede consultar información, explicar decisiones y resolver dudas sobre la situación crediticia.

Capacidad

Consulta de datos

Accede a la información del cliente directamente desde la base de datos.

Capacidad

Explicación de decisiones

Justifica por qué fue aprobado o rechazado el crédito.

Integración

IA conversacional

Demuestra cómo integrar agentes de IA con sistemas empresariales.
